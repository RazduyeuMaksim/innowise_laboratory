## Option 1: Using venv

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run the project:

   ```
   python -m my_project.main
   ```
   
---

## Option 2: Using Poetry

1. Navigate to the `my_project` folder.

2. Install dependencies:

   ```
   poetry install
   ```

3. Run the project:

   ```
   poetry run python main.py
   ```
