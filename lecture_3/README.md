### Description
A simple interactive Python program to manage student grades.  
It lets you:

- Add new students  
- Enter multiple grades for each student  
- Generate reports with averages  
- Identify the top performer  

---

### Profile output
The program prints reports like:

- Each student's average grade (or "N/A" if no grades)  
- Maximum, minimum, and overall average of all students  
- Name and average of the top student
